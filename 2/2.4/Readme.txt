Відкриваємо термінал і виконуємо команду:

	docker pull vert1cal/todo_app
 
 Після чого запускаємо контейнер командою: 

	docker run -it -d -p 3000:3000 -p 3010:3010 <my_node_app>

де <my_node_app> - назва контейнера, можна при старті замінити на ту яка вам більше подобається, 
порт 3000 в моєму випадку це сервер який віддає фронтову частину та 3010 порт відповідає за серверну частину,
якщо ви змінили значення портів  в .env на інші значення то при запуску контейнера замініть порти на ваші значення
